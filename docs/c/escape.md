sidebar_label: Escape Sequences
---

| Sequence    | Meaning |
| ----------- | ----------- |
| '\a'        | Alert(ANSI C).|
| '\b'        | Backspace.  |
| '\f'        | Form feed.  |
| '\n'        | Newline.  |
| '\r'        | Carriage return.|
| '\t'        | Tab.  |
| '\v'        | Vertical tab.  |
| '\\'        | Backslash.  |
| '\''        | Single quote(').  |
| '\"'        | Double quote(").  |
| '\?'        | Question mark.  |
| '\0oo'      | Octal value(o represents a octal digit).  |
| '\xhh'      | Hexadecimal value(h represents a hexadecimal digit).  |
